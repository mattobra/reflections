Question: What happens when you initialize a repository? Why do you need to do it?
		
Answer:  A .git file is created. It is needed to use the git commands and github.

Question: How is the staging area different from the working directory and the repository? What value do you think it offers?

Answer: Staging area - temporary folder to add files that you will eventually want to commit into the working directory
		working directory - final place to add files that you want to commit
